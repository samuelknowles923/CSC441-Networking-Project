# CSC441-Networking-Project
This is our final project for CSC 441 Networking and Data Communications. It is a program for anomaly detection in network traffic using ML algorithms.

Data files are sourced from Kaggle posted by user "Anush" anushonkar. 
https://www.kaggle.com/datasets/anushonkar/network-anamoly-detection?resource=download&select=Network+Anamoly+Detection.docx

Data consists of two files Test.txt and Train.txt stored in the data folder.
